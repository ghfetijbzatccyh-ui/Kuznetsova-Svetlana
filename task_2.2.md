# 2.2 Тема: Публикация расписания преподавателей с использованием CMS Grav

# 1. Анализ программного решения сайта кафедры ИТиЭО

В ходе выполнения работы был проведён анализ официального сайта кафедры информационных технологий и электронного обучения (ИТиЭО) https://ict.herzen.spb.ru/

В рамках анализа были рассмотрены следующие функции сайта:

* представление информации о кафедре и направлениях подготовки; 
* публикация учебных материалов и новостей; 
* отображение структуры образовательного процесса; 
* предоставление информации о преподавателях и дисциплинах.
  
На основе анализа определено, что одной из ключевых задач сайта является представление расписания и учебной нагрузки преподавателей, что требует удобной системы управления контентом.

# 3. Выбранное направление и плагин

**Направление:**
* Публикация расписания преподавателей
  
**Выбранный плагин:**
* Events
  
**Обоснование выбора:**
  
Плагин Events позволяет создавать структурированные записи событий, которые могут использоваться как учебные занятия, пары и расписание преподавателей. Он интегрируется с системой Grav и не требует сложной настройки.

# 4. Установка и настройка Grav и плагина

**4.1 Установка Grav**

* Grav был установлен в локальную директорию:

* C:\xampp\htdocs\grav

* Сервер Apache запущен через XAMPP.

Доступ к системе осуществляется по адресу:

http://localhost/grav

**4.2 Установка плагина**

Плагин установлен через консоль Grav:

* php bin/gpm install events
* 
В процессе установки были автоматически загружены зависимости:

* datetools 
* translate-date
